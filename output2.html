iOS Document For Intune Release

Make sure your app is added on thehttps://appcenter.mswith the correct Store connect and User Group in which it needs to be released.

Take the API Token, Organization name, Store name and App name that is registered on the appcentre.ms.

Now update the files (appcentre.sh, fastlane, Gemfile and .gitlab-ci.yml) according to the above information and add it in the project folder.


Now push the code on the branch which is mentioned in the yml file and you can see the pipeline running under the CI/CD Section on gitlab.


NOTE: For MAM and MDM approach separately, We need to release the app manually first time and then update the settings in the properties of app on thehttps://endpoint.microsoft.com/
